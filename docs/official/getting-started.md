# Getting Started

Welcome to the NM i AI 2026 documentation. The competition runs from **March 19 at 18:00 CET** to **March 22 at 15:00 CET** — 69 hours of non-stop AI challenges with a **1,000,000 NOK** prize pool.

## The Three Tasks

NM i AI 2026 consists of three independent AI challenges. Your overall score is the average of your normalized scores across all three tasks (33% each).

<div class="table-scroll-wrapper">

| Task                   | Type                           | Submission           |
|------------------------|--------------------------------|----------------------|
| **Tripletex**          | AI accounting agent            | HTTPS API endpoint   |
| **Astar Island**       | Norse world prediction         | REST API predictions |
| **NorgesGruppen Data** | Grocery shelf object detection | Code upload (ZIP)    |

</div>

## Quick Start

1.  Sign in at [app.ainm.no](https://app.ainm.no) with Google
2.  Create or join a team (max 4 members)
3.  Complete Vipps verification for prize eligibility
4.  Explore the task documentation in the sidebar
5.  Submit to as many tasks as you can
