# Getting Started - NM i AI 2026

## Competition Overview
The Norwegian AI Championship 2026 runs from **March 19 at 18:00 CET to March 22 at 15:00 CET** (69 hours total) with a **1,000,000 NOK prize pool**.

## Three Tasks
The competition consists of three independent challenges, each worth 33% of the overall score:

1. **Tripletex** - AI accounting agent (submit HTTPS API endpoint)
2. **Astar Island** - Norse world prediction (REST API predictions)
3. **NorgesGruppen Data** - Grocery shelf object detection (code upload as ZIP)

## Quick Start Steps
1. Sign in at app.ainm.no using Google credentials
2. Create or join a team (maximum 4 members)
3. Complete Vipps verification to be eligible for prizes
4. Review task documentation in the sidebar
5. Submit solutions to as many tasks as possible

## MCP Server Integration
Connect the documentation server to Claude Code for AI-assisted development:
```
claude mcp add --transport http nmiai https://mcp-docs.ainm.no/mcp
```
