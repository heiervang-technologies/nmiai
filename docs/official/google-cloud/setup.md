# Google Cloud Account Setup - NM i AI 2026

## Log In Steps

1. **Open a new Chrome profile** or incognito window to keep your competition account separate from personal Google accounts
2. **Navigate to console.cloud.google.com**
3. **Sign in** using your @gcplab.me email and password
4. **Select your project** from the project dropdown menu

## Cloud Shell

Cloud Shell provides a free, built-in terminal environment with essential development tools pre-installed:

- Linux VM with 5 GB persistent home directory
- Python 3, Node.js, Go, Java
- Docker for container images
- Pre-authenticated gcloud CLI
- No compute charges

Access it by clicking the terminal icon in the top-right corner of the Google Cloud console.

## Cloud Shell Editor

For a full IDE experience, click "Open Editor" in Cloud Shell or visit ide.cloud.google.com. This VS Code-based browser editor includes:

- File explorer and terminal
- Gemini Code Assist (AI coding companion)
- Direct access to Cloud Shell files

## Gemini Tools Available

| Tool | Location | Function |
|------|----------|----------|
| **Gemini Code Assist** | Cloud Shell Editor | IDE-integrated AI coding support |
| **Gemini CLI** | Cloud Shell terminal | Type `gemini` for command-line AI assistance |
| **Gemini Cloud Assist** | Console sidebar | GCP services and configuration help |
| **AI Studio** | aistudio.google.com | Direct Gemini model experimentation |

Enable Gemini Cloud Assist by clicking the Gemini icon in the console and selecting "Enable."
