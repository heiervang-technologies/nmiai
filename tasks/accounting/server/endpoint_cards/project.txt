GET /project/{id} — Find project by ID.
  Required params: id (path, REQUIRED)

  Required params: id (path, REQUIRED)
  Body: {id: integer, version: integer, changes: array, url: string, name: string, number: string, displayName: string, description: string, projectManager: Employee, department: Department, mainProject: Project, startDate: string, endDate: string, customer: Customer, isClosed: boolean, isReadyForInvoicing: }

  Required params: id (path, REQUIRED)
